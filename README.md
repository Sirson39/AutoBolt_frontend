# AutoBolt Frontend

React frontend for the AutoBolt vehicle parts and inventory management system.

## What is included

- Landing page with sign in and sign up entry points
- Admin, staff, and customer dashboard previews
- Staff workflow screens for registration, search, invoices, history, and reports
- Dark automotive theme with deep blue and orange accents

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually `http://localhost:5173`.
